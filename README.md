### TESTING

	1. In interactive mode, the shell should work like this:

	$ ./hsh
	($) /bin/ls
	hsh main.c shell.c
	($)
	($) exit
	$

	2. In non-interactive mode, the shell should work like this:
	echo "/bin/ls" | ./hsh
    	hsh main.c shell.c test_ls_2
    	$
    	$ cat test_ls_2
    	/bin/ls
    	/bin/ls
    	$
    	$ cat test_ls_2 | ./hsh
    	hsh main.c shell.c test_ls_2
    	hsh main.c shell.c test_ls_2
    	$
